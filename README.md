# IOTEG Landing Page

## Overview

**IOTEG** is a modern, responsive landing page built using **Next.js**, **TypeScript**, and **Tailwind CSS**, designed to showcase a technology-focused brand with a clean and minimal user experience.

The project emphasizes performance, scalability, and modern frontend practices, and is deployed on **Vercel (serverless hosting)** for fast global delivery.

This project serves as a frontend showcase demonstrating component-based design, responsive layouts, and production-ready deployment.
